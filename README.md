Este projeto tem como objetivo desenvolver um sistema de controle baseado em sinais EMG utilizando Inteligência Artificial.

Atualmente o sistema utiliza sinais simulados para testar:

- suavização de sinais
- calibração dinâmica
- controle analógico
- redes neurais com PyTorch
- integração com jogos em Pygame

O objetivo futuro é integrar sensores EMG reais.

Tecnologias utilizadas

- Python
- PyTorch
- Pygame
- NumPy
- 
Funcionalidades atuais

- Controle de voo baseado em intensidade
- Suavização temporal do sinal
- Zona morta para estabilização
- Física de movimento
- Rede neural supervisionada
- Calibração automática do neutro

 Objetivos futuros

- Integração com EMG real
- Melhorias na IA
- Redes neurais recorrentes (LSTM/GRU)
- Multiplayer/local control
- Mais jogos controlados por EMG
